# adventofcode2021

https://adventofcode.com/2021/

## Quick Start

1. **Clone the repository and install dependencies.**

```shell
$ npm install
```

2. **Create a default config file.**

```shell
$ mkdir config
$ vi config/default.json
```

3. **Add your adventofcode cookie to the default file.**

```js
{
  "cookie": "session=..."
}
```

3. **Run exercise.**

```shell
$ node ex1/index.js
```
